Global Food Truck Menu Analysis

Objective
Karin is planning to launch an international food truck business. This project analyses a dataset of global dishes to identify a feasible, cost-effective menu using only two cooking methods and overlapping ingredients.


Key Features
- Ingredient frequency analysis
- Cooking method breakdown
- Country of origin breakdown
- Vegetarian dish distributions
- Optimisation algorithm to select best dish combination
- Pricing comparison and averages
- AI Assistance (used for debugging and coding help)

Tools Used
- Python, Pandas, Matplotlib
- Jupyter Notebook
- itertools, basic filtering and aggregation
- Google Slides (for presentation)

Project Structure
- notebooks/ - Jupyter notebooks used in the analysis
- data/ — data files
- figures/ — charts used in the final presentation
- presentation/ — PowerPoint slides
